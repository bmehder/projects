Svelte Web Component Example

MFS Countdown Element

Overview:

This is an example of using Svelte to create (highly reusable and customizable) web components. This technique allows developers to use Svelte to sprinkle components where they want in their projects instead of going nuclear with Svelte - as in using Svelte for the whole site or page.

Usage:

  1. Link to the javascript bundle either in the head element, before the closing body element, or inline in the page if you use the defer attribute in your script tag.

    <script defer src="mfs-countdown.js"></script>

  2. Display the custom element the same way you would use any other html element.

    <mfs-countdown></mfs-countdown>

  3. Optionally, you can use component properties (or props) to customize an element.

    <mfs-countdown title="You Can Put A Title Here" end="12/31/2021" ></mfs-countdown>
