The card data is stored as a JSON array in the browser's LocalStorage object.

If you want to edit a card, you can do so by manually editing the array using your browser's debugging tools.

I would recommend saving this array locally in case you accidentally click the "Clear Cards" button.

You can also use a properly formatted JSON array to populate the cards in the first place. Here is an example of a JSON array with three cards (objects). 

              [
                {
                "question": "What is the capital of Solvenia?",
                "answer": "Ljubljana"
                },
                {
                "question": "Who is buried in Grant's tomb?",
                "answer": "No one. Tombs are above ground."
                },
                {
                "question": "Who has won the most Formula 1 driver's championships?",
                "answer": "Michael Schumacher"
                }
              ]
